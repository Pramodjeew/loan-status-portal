Loan Processing Stages

  1. Stage 1: Application Submission (අයදුම්පත් ඉදිරිපත් කිරීම)
  2. Stage 2: Documentation Verification (ලේඛන සත්‍යාපනය)
  3. Stage 3: Credit Evaluation (ණය ඇගයීම)
  4. Stage 4: Loan Underwriting (ඇපකර ඉදිරිපත් කිරීම)
  5. Stage 5: Loan Approval and Disbursement (ණය අනුමත කිරීම සහ ලබා දීම)
  6. Stage 6: Loan Servicing (ණය සේවා සැපයීම)
